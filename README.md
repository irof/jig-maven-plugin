jig-maven-plugin
============================================================

[JIG](https://github.com/dddjava/jig)

# usage

```pom.xml
<project>
    ...
    <build>
        ...
        <plugins>
            ...
            <plugin>
                <groupId>com.github.irof</groupId>
                <artifactId>jig-maven-plugin</artifactId>
                <version>0.1</version>
            </plugin>
        </plugins>
    </build>
</project>
```

```
$ mvn jig:jig
```
